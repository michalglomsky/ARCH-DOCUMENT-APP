📖 Manual Redaction Tool — User Manual

1 · Loading documents

Type a path in the PDF file or directory field and click Load, or use 📂 to browse. You can load a single PDF or an entire folder — all PDFs appear in the file list on the left. Click any filename to open it in the editor.

Load — renders the PDF at the given path (or all PDFs in the folder) and adds them to the file list.
Browse — opens a filesystem explorer with multi-select. Click a file to select it (blue highlight), Ctrl+click to toggle, Shift+click for a range, double-click to load one immediately. Click ✓ Load selected (N) to add all selected files, or navigate to a folder and click Select this folder to load the whole directory.

2 · Drawing redaction boxes

Select Draw mode, then click and drag on the document to draw a box. Boxes appear red while editing and are saved as solid black in the exported PDF.

Draw mode — drag to create a new redaction box. Keyboard shortcut: D

3 · Moving & resizing boxes

Switch to Move mode, then click a box to select it. Eight handles appear around it — drag any handle to resize, drag the box body to reposition.

Move mode — click to select a box, drag body to move, drag corner/edge handles to resize (like resizing images in Word). Keyboard: M

4 · Erasing boxes

Erase mode — click any existing box to delete it instantly. Supports undo. Keyboard: E

5 · Undo & redo

Undo — reverses the last draw, move, resize or delete. Up to 60 steps. Keyboard: Ctrl+Z
Redo — re-applies an undone action. Keyboard: Ctrl+Y

6 · Zoom

Zoom out / in by 25% per click. Keyboard: - / +
Zoom presets — jump directly to a fixed zoom level. Default is 50%.
Fit — scales the page to fill the editor width.

7 · Clearing zones

Clear page — removes all boxes on the current page (undoable).
Clear all — removes all boxes on every page of the current document (asks for confirmation).

8 · Template system

Templates let you draw boxes once and stamp them across many documents. Click Template ▾ to open the menu:

Opens the template menu. A badge appears on the button showing how many zones are saved in the current template.
  • 💾 Save as template — captures all zones from the current document. Also records which document the template came from (needed for alignment).
  • 📋 Apply to this document — stamps the template onto the current document (asks for confirmation). If an alignment anchor is set, zones shift to match.
  • 📋 Apply to all documents — stamps the template onto every document in the file list. Shows aligned vs. direct-stamp counts in the status bar.
  • ✕ Clear template — discards the saved template.

9 · Intelligent alignment anchor (per page)

Each page in the template can have its own independent anchor. Navigate to the page you want, then use 🎯 Set anchor for this page from the Template menu. Draw a box around a text label that appears consistently in that page position across all your documents, type the text, and confirm. Repeat on each page that needs alignment.

  • The canvas enters a blue crosshair mode — draw the box, type the anchor text, click ✓ Set anchor.
  • The anchor for the current page is shown as a blue dashed box with an ⚓ icon. Switching pages shows that page's anchor (if set).
  • Pages without an anchor have their zones copied directly without any shift.
  • Each page's shift is computed independently — different layouts on different pages are handled correctly.
  • If the anchor text cannot be found in a target document on a given page, those zones are stamped without shifting.
  • Separate multiple candidate strings with a comma: wniosek, Wnioskodawca
  • Use ✕ Clear this page's anchor from the Template menu to remove the anchor for the current page only.
Set anchors on each page individually when pages have different layouts. The Template badge shows the total number of anchors set.

10 · Zones panel

The right panel lists every box on the current page. Click a zone row to select/highlight it on the canvas. Click ✕ on a row to delete that specific box.

11 · Saving redacted files

Save — applies all zones on the current document and writes the redacted PDF to the Output directory.
Redact & Save All — processes every document in the file list that has at least one zone. Shows a summary when done.
Redacted areas are filled with permanent black in the saved PDF. The original files are never modified — output always goes to the Output directory.

12 · File list management

document.pdf
The active document is highlighted in orange. Click any filename to switch to it.

Each file shows its zone count (or done / error after saving). The ✕ button on each row removes the document from the session without deleting the file from disk.

13 · Session persistence

All zones, file paths, the template and the alignment anchor are automatically saved to your browser's local storage after every change. Refreshing or closing the tab preserves your work — the session is restored on the next visit.

New session — clears everything (files, zones, template, anchor) and starts fresh. Asks for confirmation before clearing.

14 · Keyboard shortcuts summary

  • D — Draw mode
  • M — Move mode
  • E — Erase mode
  • Ctrl+Z — Undo
  • Ctrl+Y — Redo
  • + / - — Zoom in / out
  • Esc — Close any open modal or browser dialog
